## 5. 즉시 적용 개선안

### [I-1] GPT 과매수 경고 시 BUY → HOLD 강제 전환 ✅ 완료

**기대 효과:** 정답률 +17~18%p (50% → 67%)

**구현 위치:** `src/koreanstocks/core/engine/analysis_agent.py` — action 정규화 직후, target_price 일관성 보정 이전

```python
# action 정규화 후 과매수 모순 차단
_weakness = result.get('weakness', '')
_weakness_text = ' '.join(str(w) for w in _weakness) if isinstance(_weakness, list) else str(_weakness)
if '과매수' in _weakness_text and result['action'] == 'BUY':
    result['action'] = 'HOLD'
    result['action_override'] = 'RSI 과매수 구간 경고 — BUY→HOLD 자동 조정'
    logger.info(f"[{name}] 과매수 경고로 BUY→HOLD 전환")
```

**구현 세부:**
- weakness 필드가 리스트 또는 문자열 양쪽으로 반환될 수 있어 두 경우 모두 처리
- `action_override` 필드로 전환 이유를 기록 → 대시보드에서 확인 가능
- 이후 target_price 일관성 보정(HOLD→SELL 분기)이 정상 작동하도록 순서 유지

**검증 근거:**
- 과매수 경고 있음: 21건, 정답률 33%, 중앙값 -4.8%
- 경고 없음: 24건, 정답률 71%, 중앙값 +3.8%
- SKAI 제거 후에도 동일 격차 유지 (30% vs 68%) — 강건한 신호

### [I-2] 거래량 폭증(6x+) 종목 추천 제외 ✅ 완료

**기대 효과:** 실패 케이스 제거, 추천 품질 안정화

**구현 위치:** `src/koreanstocks/core/engine/recommendation_agent.py` — 모듈 레벨 필터 함수

```python
def _is_volume_overheated(analysis: Dict[str, Any]) -> bool:
    """거래량 폭증(평균 대비 6배+) 여부."""
    stats = analysis.get('stats') or {}
    avg_vol = stats.get('avg_vol') or 0
    cur_vol = stats.get('current_vol') or 0
    if avg_vol <= 0:
        return False
    return (cur_vol / avg_vol) >= 6.0
```

**검증 근거:** 6x+ 종목 6건 정답률 33%, 중앙값 -3.8%. 대표 실패: 팜스토리(-2.8%, -4.8%), S-Oil(-9.4%), 대성에너지(-10.6%)

### [I-3] 과열 조합 필터 (당일 급등 + 강감성 동시) ✅ 완료

**기대 효과:** 재료소진 패턴 차단

**구현 위치:** `src/koreanstocks/core/engine/recommendation_agent.py` — 모듈 레벨 필터 함수

```python
def _is_price_overheated(analysis: Dict[str, Any]) -> bool:
    """당일 급등(5%+) + 강긍정 감성(50+) 동시 조건."""
    change_pct = abs(float(analysis.get('change_pct') or 0))
    sentiment  = float(analysis.get('sentiment_score') or 0)
    return change_pct >= 5.0 and sentiment >= 50
```

**검증 근거:** 흥구석유(감성 75, 당일 +10.1%) → -20.5%, SK증권(감성 70, 당일 +10.8%) → -8.6%

```python
def _passes_kospi_filter(analysis: Dict[str, Any]) -> bool:
    """KOSPI 종목에 황금조합 조건 강제 적용. KOSDAQ은 조건 없이 통과."""
    if analysis.get('market') != 'KOSPI':
        return True
    stats = analysis.get('stats') or {}
    avg_vol    = stats.get('avg_vol') or 1
    cur_vol    = stats.get('current_vol') or 0
    vol_mult   = (cur_vol / avg_vol) if avg_vol > 0 else 0
    sentiment  = float(analysis.get('sentiment_score') or 0)
    change_pct = abs(float(analysis.get('change_pct') or 0))
    return vol_mult < 3.0 and 0 <= sentiment <= 40 and change_pct < 3.0
```

**검증 근거:** KOSPI 비황금조합(n=13) 정답률 23%, 중앙값 -9.4% vs 황금조합(n=4) 정답률 75%, 중앙값 +15.1%

**실동 확인 (2026-04-02, 최근 추천 18건 시뮬레이션):**
- 4건 차단 — 거래량 6x+ 3건(테라뷰, 이루온, 일지테크), KOSPI 감성 조건 1건(KEC)
- 14건 통과

**알려진 동작 특성 — KOSPI 감성 하한(0 미만) 차단:**

KEC(KOSPI) 사례: 거래량 0.4x·당일 -0.4%로 안정적이나 sentiment=-10으로 차단됨.

```
KEC: vol=0.4x ✅  |chg|=0.4% ✅  sentiment=-10 ❌ (조건: 0≤sentiment≤40)
```

원 분석의 황금조합은 "강긍정 차단"이 주목적이었으나, 현재 하한 `0`이 **약부정 감성 KOSPI 종목도 함께 차단**함. 데이터 부족(KOSPI 황금조합 n=4)으로 하한 완화 근거가 없어 보수적 기준 유지.

> **재검토 시점:** 2026-04-30 이후 KOSPI 성과 데이터(30건+) 누적 후 하한을 -25로 완화할지 검토.

### [S-3] 복합점수 → 세션 내 백분위 순위 변환 ✅ 완료

**현황:** 복합점수 64~92점에 전체 집중 → 절대 구간 분석 의미 없음

**구현 위치:** `src/koreanstocks/core/engine/recommendation_agent.py` — 품질 필터 직후

# 세션 내 백분위 계산 후 score_percentile 필드 추가 (기존 composite_score 유지)
if results:
    _all_scores = sorted(_composite_score(r) for r in results)
    _n = len(_all_scores)

def _continuity_penalty(code: str, session_date: str, db_manager) -> float:
    recent = db_manager.get_recent_recommendations(code, days=10)
